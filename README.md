# string-convert

It is

* `Prelude.String`
* `Data.ByteString.ByteString`
* `Data.ByteString.Lazy.ByteString`
* `Data.Text.Text`
* `Data.Text.Lazy.Text`

convert wrapper.

It is **simple** and **easy**.

# provide function

* `toString`
* `toByteStringStrict`
* `toByteStringLazy`
* `toTextStrict`
* `toTextLazy`

# note

It wrapper expect that `ByteString` encode is utf-8.
